# Global Video Meet with AI

## Overview

Global Video Meet with AI is a web-based video conferencing application inspired by Zoom and Google Meet. The platform enables users to create and join virtual meetings in real-time using ZEGOCLOUD's video communication services. The application provides a seamless and interactive meeting experience directly from the browser.

## Features

* Create and join video meetings instantly
* Real-time audio and video communication
* Unique Meeting ID generation
* Multi-user video conferencing
* Mute and unmute microphone
* Turn camera on and off
* Screen sharing support
* Responsive and user-friendly interface
* Browser-based access without software installation
* Powered by ZEGOCLOUD API for reliable communication

## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Video Communication

* ZEGOCLOUD Video Call API
* WebRTC

## How It Works

1. User enters a username.
2. User creates a new meeting or joins an existing meeting using a Meeting ID.
3. The application connects participants through the ZEGOCLOUD API.
4. Users can communicate through live audio and video streams.
5. Participants can manage camera and microphone settings during the meeting.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Global-Video-Meet-with-AI.git
```

2. Open the project folder.

3. Configure your ZEGOCLOUD credentials in the JavaScript file:

```javascript
const appID = YOUR_APP_ID;
const serverSecret = "YOUR_SERVER_SECRET";
```

4. Run the application using a local server.

## Installation and Setup

### Prerequisites

* Node.js (v18 or later)
* npm or yarn
* ZEGOCLOUD Account and API Credentials

### Clone the Repository

```bash
git clone https://github.com/nirmal14820/Global-Video-Meet-with-AI.git
cd Global-Video-Meet-with-AI
```

### Install Dependencies

```bash
npm install
```

### Configure ZEGOCLOUD

Add your ZEGOCLOUD credentials in the appropriate configuration file:

```javascript
const appID = YOUR_APP_ID;
const serverSecret = "YOUR_SERVER_SECRET";
```

### Start the Development Server

If the project uses Vite:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:xxxx
```

If the project uses Create React App:

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```

### Build for Production

```bash
npm run build
```

### Preview Production Build (Vite)

```bash
npm run preview
```
